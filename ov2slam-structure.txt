OV2SLAM

1. ov2slam_node launches SlamManager::run and other sensor grabber

2. SlamManager:

Creates:
- FeatureExtractor
- FeatureTracker
- MapManager (manages keyframes & map points)
- Mapper (manages keyframe processing: triangulation, map tracking, loop closing, bundle adjustment)

Run:
- grabs image
- creates new frame

3. FeatureExtractor:
- SkiTomasi corner extraction
- BRIEF descriptors

4. FeatureTracker:
- forward/backwards tracking using LK optical flow

5. Map Manager:
- create keyframe from raw image
- update frame covisibility
- extract/add/describe keypoints to frame
- compute triangulation
- add/get/update/merge/delete map point
- stereo matching

6. Mapper:

Run:
- get new KF
- stereo/temporal triangulation
- check for reset
- update frame covisibility
- matching to local map
- add new KF

Contains:
- triangulate stereo/temporal (opengv triangulation)
- merge matches
- match to map
- run full BA
